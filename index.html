Welcome to Advanced Traffic Control System

What is ATCS? ATCS is a program that can take in traffic flow data through GPS and other sources, and control city intersections based on traffic flow.

How can I get it?

If you want to experiment, click one of the download buttons above. Note that you will need to compile it yourself. The code you download will be unstable and is recommended for testing and trial purposes only.
